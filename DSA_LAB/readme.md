# 📌 Syllabus Coverage with Practice Questions

---

## 🔹 Array

### Topics Covered

* Traversal, Insertion, Deletion
* Searching: Linear Search, Binary Search
* Sorting: Bubble Sort, Selection Sort
* Array + String Problems

### Practice Questions

1. Reverse an Array
2. Check if Array is Palindrome
3. Two Sum (Sorted Array)
4. Remove Duplicates from Sorted Array
5. Move Zeroes
6. Sum of Subarray of Size K
7. Average of Subarray of Size K
8. Maximum Element in Subarray of Size K
9. Count Subarrays of Size K
10. Find All Anagrams in a String

---

## 🔹 String

### Topics Covered

* String Functions
* Character Functions
* String Traversal

### Practice Questions

1. Reverse a String
2. Check Palindrome String
3. Count Vowels and Consonants
4. Reverse Each Word of String
5. Reverse Vowels of String
6. Count Occurrence of a Character
7. Remove Spaces from String

### Topics Covered

## 🔹 Stack

### Topics Covered

* Stack Operations: Push, Pop, Peek
* Array-Based & Linked List-Based Stack
* Applications:

  * Infix to Postfix Conversion
  * Postfix Evaluation

### Practice Questions

1. Valid Parentheses
2. Remove All Adjacent Duplicates in String
3. Baseball Game (Stack Operations)
4. Reverse a String using Stack
5. Next Greater Element
6. Previous Smaller Element

## 🔹 Tree

### Topics Covered

* Basic Terminology
* Tree Traversals:

  * Preorder
  * Inorder
  * Postorder
* Binary Search Tree

### Practice Questions

1. Count Total Nodes in a Binary Tree
2. Height (Max Depth) of Binary Tree
3. Print Preorder Traversal
4. Print Postorder Traversal
5. Check if Tree is Same as Another Tree
6. Print Inorder Traversal

# 📝 Important Notes

* Focus on traversal order carefully
* Dry run tree diagrams before coding
* Practice recursion base conditions
* Understand node visiting sequence
* Write clean and modular code
