Two Pointer ek technique hai jo mostly arrays aur strings ke
questions solve karne ke liye use hoti hai. Isme 
hum ek ke badle do pointers (indexes) use karte hain.
Isse time complexity kam ho jati hai aur code efficient ho jata hai.


1. Pointer kya hota hai

Pointer simple language me array ka index hota hai jo kisi position ko point karta hai.
arr = [2, 4, 6, 8, 10]
       0  1  2  3   4
Agar pointer i = 2 hai
to vo 6 ko point kar raha hai.


2. Two Pointer kya hota hai

Jab hum ek hi array me do indexes use karte hain problem
solve karne ke liye, use Two Pointer Technique bolte hain.
arr = [1,2,3,4,5]

left  -> 0
right -> 4
1 2 3 4 5
L       R


Yaha
left ek pointer hai
right dusra pointer hai
